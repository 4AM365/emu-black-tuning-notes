# Other — settings reference

> **Software page:** *Other*. Full symbol catalog: [tune_feature_tree.md → Other](tune_feature_tree.md).

Base/system settings that do not belong to a feature page — engine setup (cylinders, displacement, firing order, torque model), start/stop control, checksums, shared axis bins, and debug/misc flags.

This is largely **configuration**, not tuning-principle territory — the exhaustive symbol list is in the catalog. This page is the home for any tuning notes that arise for this feature; the sub-nodes below mirror the software tree.

## Sub-nodes

- **Engine / base setup** (18) — `engineTorqueNm`, `engineDisplacement`, `firingOrder1`, `firingOrder2`, `firingOrder3` …
- **Checksums** (2) — `tablesChecksum`, `variablesChecksum`
- **Base config / misc** (13) — `autotuneDirtyFlags`, `commentData`, `calToolState`, `debugSwitch1`, `debugSwitch2` …
- **Shared axis bins** (10) — `airflowBins4`, `airflowBins5`, `mapBins`, `mapBins8`, `mapBins8B` …
